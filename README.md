# DNA Methylation Age Prediction

## Project Structure
- `data/` - Dataset files
- `results/` - Output files and visualizations
- `methylation_age_analysis.py` - Main analysis script

## How to Run
1. Navigate to project folder
2. Run: `python methylation_age_analysis.py`

## Results
- Linear Regression MAE: ~5-6 years
- Random Forest MAE: ~3-4 years
- Improvement: ~30-40%

## Dataset
- 656 samples
- 353 CpG sites (Horvath clock)
- Age range: 19-95 years
